# Zitrone Brand Strategy

This repository captures the working documents that define the Zitrone brand strategy.
It is organized as a progressive narrative from context to decisions, with each section
containing focused, editable artifacts.

## Structure

- **01-context/**: External market context and competitive landscape.
- **02-insight/**: Insights about buyers, decision-making, and tensions.
- **03-positioning/**: Positioning, differentiation, and clarifying boundaries.
- **04-messaging/**: Narrative, framing, and proof points.
- **05-experiments/**: Messaging and positioning tests.
- **decisions/**: Time-stamped decisions for historical reference.

## How to use

1. Start in `01-context/` to gather facts and references.
2. Move to `02-insight/` to translate observations into actionable insights.
3. Draft positioning artifacts in `03-positioning/`.
4. Build and validate messaging in `04-messaging/`.
5. Capture experiments and learning in `05-experiments/`.
6. Record durable decisions in `decisions/`.

## Contribution notes

- Keep entries concise and actionable.
- Add citations or links where possible.
- Use clear, declarative language.
